# UPS Sort Simulator

A compact, one-page, mobile-friendly browser simulator for fictional UPS-style sort operations decision-making.

Every action advances a 5-hour sort clock by 15 minutes, followed by a 15-minute wrap-up. Decisions affect safety, productivity, service, team morale, accuracy, leadership pressure, required task completion, and the final 0-100 score.

> Disclaimer: No real UPS data is used. Scenarios are fictional and simplified for training/gameplay. This independent project is not an official UPS tool and is not affiliated with, endorsed by, or sponsored by UPS.

Made by [Brett Jessen](https://www.brettjessen.com).

## Run Locally

Open `index.html` in any modern browser.

No backend and no install step required.

## Deploy on GitHub Pages

1. Create or open your GitHub repository.
2. Upload these files.
3. Go to **Settings → Pages**.
4. Select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
